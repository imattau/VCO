# @vco/vco-sync

Range-based set reconciliation for the Verifiable Content Object protocol.

## Features

- **Efficient Reconciliation**: Synchronize large data sets with minimal network traffic.
- **Merkle Proof Integration**: Cryptographic proofs for range-based comparisons.
- **Sync State Machines**: Robust reconciliation protocols for unreliable connections.
- **Protocol Agnostic**: Can be used over any transport layer.

## Installation

```bash
npm install @vco/vco-sync
```
