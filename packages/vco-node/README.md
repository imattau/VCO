# @vco/vco-node

A lightweight, high-performance Node.js implementation of the Verifiable Content Object protocol.

## Features

- **CLI Tooling**: Command-line interface for VCO operations.
- **Embedded Relay**: Can be used to build custom relay or client logic.
- **Channel Sync**: Automatic synchronization of content for subscribed channels.
- **Persistent Cache**: Local cache for fast retrieval and reconciliation.

## Installation

```bash
npm install @vco/vco-node
```
