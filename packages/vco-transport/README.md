# @vco/vco-transport

Transport layer and session management for the Verifiable Content Object protocol.

## Features

- **Libp2p Adapters**: Standardized adapters for peer-to-peer communication.
- **TOL (Transport Obfuscation Layer)**: Built-in Noise padding and session management.
- **Packetization**: Handles large payload transport and reassembly.
- **Multiplexing**: Supports Yamux and QUIC stream management.

## Installation

```bash
npm install @vco/vco-transport
```
