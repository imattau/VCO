# @vco/vco-schemas

Standard protocol schemas and encoders for the Verifiable Content Object protocol.

## Features

- **Social Media Schemas**: Standardized formats for Posts and Profiles.
- **Manifest Support**: Sequence and collection manifests for large data sets.
- **Strict Validation**: Protobuf-backed schemas with runtime validation.
- **Interop Ready**: Cross-application compatibility for verifiable content.

## Installation

```bash
npm install @vco/vco-schemas
```
