# @vco/vco-core

The core logic of the Verifiable Content Object protocol.

## Features

- **Envelope Management**: Construct and validate VCO envelopes.
- **Proof-of-Work**: Built-in PoW solver and verifier for rate-limiting.
- **ZKP Integration**: Extension points for privacy-preserving authentication.
- **Protobuf Support**: Seamless encoding/decoding of protocol buffers.

## Installation

```bash
npm install @vco/vco-core
```
