# @vco/vco-crypto

Cryptographic primitives for the Verifiable Content Object protocol.

## Features

- **Noble-based Implementation**: Built on top of high-performance `@noble` curves and hashes.
- **Ed25519 Support**: Fast signatures and multikey derivation.
- **Blake3 Hashing**: Protocol-standard hashing for envelopes and payload verification.
- **Pluggable Provider**: Core package remains crypto-library agnostic.

## Installation

```bash
npm install @vco/vco-crypto
```
